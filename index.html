Easy Installation Guide

  1. 1

    Download a Launcher

    Use a modern launcher like Modrinth App or Prism Launcher. They make installing mods simple.

  2. 2

    Install Your Modpack

    Use the launcher's search function to find and install your chosen modpack (e.g., "Fabulously Optimized") for the latest Minecraft version with one click.

  3. 3

    Allocate RAM

    In the launcher's instance settings, set the allocated RAM. 4GB (4096MB) is a great starting point for most low-end PCs.

RAM Allocation Guide

Allocate enough RAM, but not too much. Over-allocating can hurt performance.
